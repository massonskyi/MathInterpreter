Документация Mathematic Interpreter

Работа с переменными

Создание и использование переменных в Mathematic Interpreter интуитивно понятно и следует стандартным математическим обозначениям.

# Create variable 
x = 2

# Print variable x
x
2

Работа с векторами

Векторы в MI представляют собой упорядоченные наборы чисел. Поддерживаются все основные операции линейной алгебры.

# Create Vector
V = [1,2,3,4,5,6,7,8,9]
V
[1, 2, 3, 4, 5, 6, 7, 8, 9]

Векторы можно комбинировать со скалярными величинами:

# Summary vector with variable
new_v = V + 10
new_v
[11, 12, 13, 14, 15, 16, 17, 18, 19]

Работа с матрицами

Матрицы представляют собой двумерные массивы чисел. MI предоставляет широкий набор операций для работы с ними.

# Create Matrix
M = [[1,2,3],[1,2,3]]
M
[[1, 2, 3], [1, 2, 3]]

Операции с матрицами:

# Summary matrix with variable
new_m = M + 10
new_m
[[11, 12, 13], [11, 12, 13]]

Транспонирование матрицы:

# Transpose matrix
T(new_m)
[[11, 11], [12, 12], [13, 13]]
